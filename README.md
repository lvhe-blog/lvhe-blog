### 2024.8.17
上
七月八月蚊当家，
飞到耳边嗡嗡哇；
不知其等是啥咖，
一口一个小红疤。
下
熟知禅师不解意，
撑得来客肚似瓜；
清晨观诸皆故去，（慈悲蚊子，不能让它们尽其贪心，否则，蚊子贪心不足吸血太饱，会全部死掉）
小鸟一来全归它。
